# Store Rating Web Application

## Overview
A full-stack web application that allows users to register, log in, browse stores, and submit ratings. The system supports role-based access for administrators, store owners, and users, with secure authentication and MySQL database integration.

## Features
- User Registration and Login
- Role-Based Access Control
- Browse and Search Stores
- Submit Ratings and Reviews
- Admin Dashboard
- Store Owner Dashboard
- Secure Authentication
- RESTful APIs
- MySQL Database Integration

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Store-Rating-Web-Application.git
```

### Install dependencies

```bash
npm install
cd backend
npm install
cd ../frontend
npm install
```

### Start the application

```bash
npm run dev
```
